# 🏟️ SportNest - Sports Facility Booking System

SportNest is a modern full-stack sports facility booking platform built with **Next.js**. It allows users to explore sports facilities, view detailed information, book available time slots, and manage their bookings through a responsive and user-friendly interface. The platform also includes role-based authentication and an admin dashboard for managing facilities and bookings.

## 🚀 Getting Started

### Clone the repository

```bash
git clone YOUR_CLIENT_REPOSITORY_URL
```

### Navigate to the project

```bash
cd sportnest-booking-system
```

### Install dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env.local` file in the project root.

```env
NEXT_PUBLIC_API_URL=YOUR_BACKEND_URL

NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=
NEXT_PUBLIC_FIREBASE_APP_ID=
```

### Start the development server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
